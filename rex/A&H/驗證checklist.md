## ❌ 出問題了怎麼辦

### 症狀:畫面空白 / 載入卡住
**原因**:fetch `shared.js` 或 `benefitsLib.json` 失敗(沒部署 / 路徑錯)
**檢查**:F12 Network 看哪個檔 404
**Rollback**:改 `index.html` 把 url 指回 `保險建議書系統_v3_20.html`

### 症狀:畫面正常但理賠卡片全空
**原因**:`benefitsLib.json` 內容缺失或解析失敗
**檢查**:F12 Console 看 `INSURANCE_DB.benefitsLib` 是不是 `null`
**Rollback**:同上

### 症狀:保費算錯(不是 22,815)
**原因**:這個保費計算**跟 Phase 1 改動完全無關**,理論上不可能。如果真的錯了,可能是檔案沒完整下載 — 強制重新整理(Ctrl+Shift+R)
**Rollback**:若強制 reload 也不行,rollback 並回報具體錯誤金額

### 症狀:F12 Console 看到 `AHShared is undefined`
**原因**:`shared.js` 沒部署 / 路徑錯
**檢查**:確認 `shared.js` 跟 HTML 在**同一個目錄**
