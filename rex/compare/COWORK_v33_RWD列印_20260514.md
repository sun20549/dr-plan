# Cowork v33 — RWD 全面強化 + 列印優化

---

## ✅ 這輪做了什麼

### 1. 全面 RWD 重寫(5 個 breakpoint)

原本只有 4 個 @media,有些重疊有些缺漏。重新設計 5 個 breakpoint:

| Breakpoint | 對應裝置 |
|---|---|
| ≥ 1280px | 桌機 |
| 1024-1279px | iPad 橫式 |
| 768-1023px | iPad 直式 |
| 480-767px | 手機橫式 / 大手機 |
| ≤ 479px | 手機直式 |

### 2. 各裝置的調整

**iPad 直式(768-1023)**:
- 表單從 3 欄 → 2 欄
- card padding 縮 28×24px
- product-table 字體微縮 12.5px
- summary-bar gap 緊縮

**手機(≤767px)**:
- Header 縮:logo 40×40,h1 18px,SAVINGS 副標 9px
- step-indicator dot 縮 26px
- card padding 18×16px(原 36×40)
- form 改單欄,gap 16px
- 按鈕全寬(.btn 100% width)
- summary-bar 改縱向堆疊
- product-table:外包 `.product-table-wrap`,可橫向滾動
- 字體 12px,padding 緊縮

**手機直式(≤479px)**:
- SAVINGS 副標完全隱藏(節省空間)
- step-line 縮到 8px
- 標題 17px

### 3. 列印優化

**@page**:
- A4 橫式
- 邊距 12mm × 10mm

**列印樣式**:
- `print-color-adjust: exact` 強制保留色塊
- 公司 tag / accuracy tag 加 `border:1px solid currentColor`(讓黑白印表機也能看出邊界)
- `page-break-inside: avoid` 防止行被切斷
- 自動加列印頁尾:**rex1688.com/rex/compare · 列印日期 YYYY/MM/DD**(每頁注入)

### 4. 修 product-table 結構
原本沒包 wrap,手機會出畫面。加 `<div class="product-table-wrap">` 包覆,搭配新 CSS 可橫向滾動。
