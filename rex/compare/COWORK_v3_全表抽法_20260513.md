## ⚠️ 關鍵卡點:**前端引擎還沒升級**

我已經把 GP+PV 全表寫進 JSON,但 `index.html` 的 `calcProductForUser` 函式還是用舊的 **twlife_v1**(線性縮放),**還沒寫 twlife_v2_full**。

**意思是**:JSON 有完整全表,但前端還在用「舊算法 + 新資料」 — 等於資料沒被用上。

要真正發揮全表精算,需要在 `index.html` 加:

```js
function calcProductTwlifeV2Full(product, user) {
  // 1. 從 user input 構造 code (例如 0641M = 6年期 41歲男)
  const period = String(product.meta.period).padStart(2,'0');
  const age = String(user.age).padStart(2,'0');
  const sex = user.sex;
  const code = period + age + sex;
  
  // 2. 查 GP 表算保費
  const gp_rate = product.gp_table[code];
  if (!gp_rate) return {error: '查無此年齡費率'};
  
  // 3. 從預算反推 SA
  const SA = (user.budget / gp_rate) * 1000;
  
  // 4. 查 PV 表算逐年保價金
  const pv_years = product.pv_table[code];
  const schedule = pv_years.map((rate, i) => ({
    y: i+1, age: user.age + i,
    cum_prem: i < period_yrs ? SA/1000 * gp_rate * (i+1) : SA/1000 * gp_rate * period_yrs,
    cv_basic: SA/1000 * rate,
    // ... 增值回饋分享金等
  }));
  
  return {sa: SA, schedule, ...};
}
```

**估時**:1~2 小時寫完整精算邏輯(含增值回饋分享金 + 高保額折扣)
