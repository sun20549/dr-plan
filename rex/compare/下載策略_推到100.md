## 檔名規範(很重要!)

**格式:** `{幣別}_{年期}yr_{年齡}{性別}_{類型}.html`

```
✅ 範例:
USD_2yr_21M_increment.html
USD_2yr_41M_increment.html
USD_2yr_30F_increment.html
USD_10yr_21M_increment.html
...
TWD_6yr_21M_refund.html       (還本型用 refund)
USD_20yr_30F_dividend.html    (分紅型用 dividend)
```

**類型代號:**
- `increment` = 美元/台幣利變非還本(增值型)
- `refund` = 美元/台幣利變還本
- `dividend` = 美元/台幣分紅
- `fixed_rate` = 預定利率(很少用)
