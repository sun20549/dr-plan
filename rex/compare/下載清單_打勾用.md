## 📦 上傳給我的方式

每下完 **5-10 個** 就:

1. 全選那批 .html + 對應 `_files` 資料夾
2. 右鍵 → **「壓縮成 zip」**
3. 命名 zip,例如:`第1批_USD_2_3_5yr.zip`
4. 打開 Cowork → 右上「上傳檔案」→ 拖 zip 進來
5. 跟我說:**「處理第 X 批」**

我會自動:
- 解壓
- 抽 cv + death 雙表(每檔 2 張表)
- 寫入對應 JSON 欄位
- 升 cache + 驗證 + push 上線
