## ✅ 重大突破:VelvetSweatshop 解開遠雄

### 試暴破發現
做字典攻擊測試時,直接驗證發現遠雄 .xls 加密的密碼是 **`VelvetSweatshop`**(這是 Microsoft Excel 的「唯讀建議」預設密碼)

之前 v15c extractor 雖然有 VelvetSweatshop 在 PASSWORD list 但因錯誤路徑沒有實際被使用。這輪修復後 — **37 個遠雄 Excel 全部解開** ✅

### 新增 farglory 格式
extractor 加第 4 種:遠雄 sheet 結構為
- 保費:條件/險別/SEX/年齡/費率/每萬
- 保單價值:per (sex, age, dur 0..N)
- 解約金:per (sex, age, dur 0..N)
- 身故保障:per (sex, age, dur 0..N)
- 跟其他公司不同:用「男/女」中文 + unit_size = 10000(每萬)

### 還改進了 index 解析
遠雄檔名常有 `(BU1+MC1+NZA)` 這種多碼合一格式,新 index 把每個 +-分隔的碼都對應到同一檔
