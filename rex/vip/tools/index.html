教學模式（每家經代網設定一次）
30 秒 / 家

這是 v3 的核心新功能 — 讓腳本「記住」每家經代網的對帳單在頁面的哪個位置。

到每家經代網的對帳單頁面執行**一次**：

  1. 登入經代網，導到任一客戶的對帳單頁
  2. 點右下角浮動按鈕 → 點 「🎓 教學：標出對帳單位置」（或按 Alt + T）
  3. 整個頁面進入教學模式，畫面變黃色提示「🎓 教學模式 ・ 把滑鼠移到對帳單區塊上」
  4. 滑鼠移到「對帳單核心區塊」上 → 該區塊會浮現金色外框 → 點擊它
  5. 跳通知「✓ 已學會 xxx.com.tw 的對帳單位置」 → 完成！
🎯 怎麼選對「對帳單區塊」？
建議選對帳單**外層的容器**（含表格 + 金額 + 日期那一整塊），不要只選表格本身。
選完之後右下角的浮動按鈕會變綠色✨ —「已就緒」，代表這家網站已學會。

4 家經代網都要各教一次（共 4 次設定，每次 30 秒）。設定完成後，未來都不用再教，瀏覽器永久記住。

💡 教學記憶儲存在哪？
每家網站的 CSS 選擇器儲存在 localStorage，跟著你的瀏覽器設定走。如果你清掉瀏覽器資料就要重新教學一次。

疑難排解

Q: 按了截圖按鈕沒反應？
A: 看右下角是否有出現浮動按鈕；沒有的話開瀏覽器 DevTools (F12) → Console，看有沒有錯誤訊息。

Q: 截圖結果空白 / 只截到一半？
A: 經代網用了 iframe，html2canvas 無法穿透 iframe。臨時解法：用 Windows 內建 Win+Shift+S 截 iframe 內容，搭配 paste-tool.html 貼上存檔。

Q: 「📁 連結資料夾」按了沒反應？
A: 必須是 Chrome 或 Edge 桌面版（Firefox / Safari 不支援 File System Access API），自動降級為下載到 Downloads 資料夾，再手動拖到 cases 資料夾。

Q: 按鈕被經代網的東西擋到了？
A: 腳本 z-index 已設最高 (2147483647)，理論上不會被擋。若還是有，請截圖告訴我，我針對該頁加調整。

⚠ 法律免責
這個工具僅在你本人帳號內、本人權限範圍內的頁面執行截圖。請確保遵守各經代網的使用條款，且截圖內容已適度處理個資（個人姓名、身分證號等）後才可上傳到公開網站。
