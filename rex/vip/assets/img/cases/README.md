# 案例對帳單截圖目錄

## 檔名規則（含年月，自動分組顯示）

**檔名格式**：`{caseId}-{YYYYMM}-{n}.{ext}`

| 案例 | 年月 | 範例 |
|---|---|---|
| U0033（2025/05 第 1 張） | 202505 | `u0033-202505-1.jpg` |
| U0033（2025/05 第 2 張） | 202505 | `u0033-202505-2.jpg` |
| U0033（2025/06 第 1 張） | 202506 | `u0033-202506-1.jpg` |
| K3007（2026/02 第 1 張） | 202602 | `k3007-202602-1.jpg` |

**3 個規則**：
- **小寫**案例編號（u0033 ✓，U0033 ✗）
- 年月用 **6 位數**：YYYYMM（例如 2025 年 5 月 = `202505`）
- 序號**從 1 開始，連續編號**（同月份的第 1、2、3...張）

## 網站如何顯示

網站會：
1. **自動依年月分組** → 不同月份的對帳單分開顯示
2. **最新月份置頂** → 帶綠色「最新」徽章 + 金色邊框
3. **舊月份依時間倒序排列** → 邊框轉灰、半透明（但仍可點擊放大）
4. **點任一張圖** → Lightbox 全螢幕檢視（左右切換、ESC 關閉）

## 支援格式

- `.jpg` / `.jpeg`（推薦）
- `.png`
- `.webp`

## 舊格式相容

舊格式 `u0033-1.jpg`（沒年月）**仍可使用**，會以「無日期」分組顯示。
建議所有新上傳的圖都使用含年月的新格式。

## 工具推薦

不必手動命名 — 用 **paste-tool.html** 或 **Tampermonkey 截圖助手** 會自動產生正確檔名：
- [paste-tool.html](../../../paste-tool.html) — 截圖貼上工具（用於 Win+Shift+S 截圖）
- [Tampermonkey 截圖助手](../../../tools/) — 直接在經代網頁上一鍵截圖

兩個工具都已支援「建置月份」輸入，預設本月。

## 圖片壓縮建議

- 寬度 ≥ 1200px（網頁會自動縮放）
- 檔案大小盡量 < 200KB（用 [TinyPNG](https://tinypng.com) 或 [Squoosh](https://squoosh.app)）
- 個資（身分證、姓名、電話）請先打碼

## 累積保留

每月新上傳會與舊圖**累積**，不會覆蓋舊月份。
讓客戶看到「這個案例從 5 月到 8 月的對帳單變化」→ 說服力最強。
