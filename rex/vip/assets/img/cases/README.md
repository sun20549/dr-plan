# 案例對帳單截圖目錄

## 檔名規則（含年月日，精確到「天」）

**檔名格式**：`{caseId}-{YYYYMMDD}-{n}.{ext}`

| 案例 | 年月日 | 範例 |
|---|---|---|
| C0235（2025/05/13 第 1 張） | 20250513 | `c0235-20250513-1.jpg` |
| C0235（2025/05/13 第 2 張） | 20250513 | `c0235-20250513-2.jpg` |
| C0235（2025/06/14 第 1 張） | 20250614 | `c0235-20250614-1.jpg` |
| K3007（2026/02/05 第 1 張） | 20260205 | `k3007-20260205-1.jpg` |

**3 個規則**：
- **小寫**案例編號（u0033 ✓，U0033 ✗）
- 年月日用 **8 位數**：YYYYMMDD（例如 2025 年 5 月 13 日 = `20250513`）
- 序號**從 1 開始**（同日的第 1、2、3...張）

## manifest.json — 加速秒查

為避免網站要試探 4000+ 種檔名（年×月×日×序號），這個資料夾還有一個 **manifest.json**，記錄所有圖檔清單：

```json
{
  "c0235": [
    "c0235-20250513-1.jpg",
    "c0235-20250513-2.jpg",
    "c0235-20250614-1.jpg"
  ],
  "u0033": [
    "u0033-20260514-1.jpg"
  ]
}
```

**這個檔案會自動維護** — paste-tool 或 Tampermonkey 每次存圖都會更新它，你不用手動編輯。

如果手動加圖到資料夾，到 paste-tool 點底部的「🔧 重建檔案清單」即可重建。

## 網站如何顯示

網站會：
1. **讀 manifest.json**（1 個請求，秒查完所有圖）
2. **依年月分組**顯示（同月的圖放一起）
3. 每張卡片頂部薄條顯示「1/3 ・ 2025/05/13」
4. 點任一張 → Lightbox 全螢幕（左右切換 / ESC 關閉）

## 支援格式

- `.jpg` / `.jpeg`（**推薦**，paste-tool 預設）
- `.png`
- `.webp`

## 圖片壓縮建議

- 寬度 ≥ 1200px（網頁會自動縮放）
- 檔案 < 200KB（paste-tool 預設 JPEG 85% 壓縮）
- 個資（身分證、姓名、電話）請先打碼
